# Pascal's Triangle II (Easy)

## Link

[Pascal's Triangle II](https://leetcode.com/problems/pascals-triangle-ii/)

## Question

> Given an index *k*, return the *k*th row of the Pascal's triangle.
> 
> For example, given *k* = 3,
> Return `[1,3,3,1]`.
