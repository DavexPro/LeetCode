# Plus One (Easy)

## Link

[Plus One](https://leetcode.com/problems/plus-one/)

## Tips

这个题就是实现大数（用数组存储）加1的功能。

需要注意的就是满十进一这个规则，以及类似999+1=1000需要在第一位添一个1这种特例。

比方说：

> 99999 = [9,9,9,9,9]
>
> 99999 + 1 = [1,0,0,0,0,0]
>
> 99998 + 1 = [9,9,9,9,9]
